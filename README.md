# vog-cluster

NATS protocol contracts for the vog horizontally-scaled game cluster.

Provides typed messages, subject constants, and encoding helpers used by
vog-spaces (coordinator), vog-game (game instances), vog-lobby (gateway),
vog-games (rating publisher), and vog-tourneys (tournament events).

See [the architecture spec](https://github.com/khorost/vog-arch/blob/main/docs/superpowers/specs/2026-04-13-horizontal-scaling-design.md) for context.

## Status

Work in progress. See implementation plan in vog-arch.
